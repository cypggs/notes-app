## ⚠️ 下一步操作（重要！）

在浏览器打开应用之前，你需要先设置数据库：

### 1️⃣ 执行数据库 SQL

1. 打开浏览器，访问：
   ```
   https://supabase.com/dashboard/project/mclpscvtkxldycxoidoc
   ```

2. 点击左侧菜单的 **SQL Editor**

3. 点击 **New query** 创建新查询

4. 打开项目中的 `database.sql` 文件，复制所有内容

5. 粘贴到 SQL 编辑器，点击 **Run** 执行

6. 等待执行完成（应该几秒钟就完成）

### 2️⃣ 验证数据库

执行完成后，在 Supabase Dashboard 中：

1. 点击左侧 **Table Editor**
2. 应该能看到 3 个表：
   - `notes` (包含 2 条示例笔记)
   - `tags` (包含 4 个标签)
   - `note_tags` (笔记和标签的关联)

### 3️⃣ 访问应用

打开浏览器访问：
```
http://localhost:3000
```

## ❓ 常见问题

**Q: 页面显示"加载中..."不消失？**
A: 检查是否已在 Supabase 执行 SQL 脚本

**Q: 编辑器对话框不显示？**
A: 刷新页面重试，检查浏览器控制台是否有错误

**Q: 主题切换不生效？**
A: 清除浏览器缓存后重试

**Q: 如何添加新标签？**
A: 当前版本使用预设标签，可以在 Supabase Table Editor 中的 `tags` 表手动添加
